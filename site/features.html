Features

An in-depth look at everything Ammonite can do.

Moderation

A full suite of moderation tools for managing your community.

Commands

  • /ban — Permanent or timed bans
  • /kick — Kick members instantly
  • /mute — Timed mutes with duration
  • /timeout — Discord native timeout
  • /warn — Warning system with history
  • /purge — Bulk message deletion
  • /lockdown — Channel lock/unlock
  • /softban — Ban + unban to purge messages

Moderation History

Every moderation action is logged and reviewable with /hist <user>. Warnings can be viewed with /warns [user] and removed with /unwarn.

Punishment DM Notifications

When enabled via /punishment-dm, users receive a DM when warned, muted, banned, or kicked. Users can also appeal punishments through the built-in appeal system.

Economy & Banking

A full virtual currency system with earning, spending, gambling, and banking.

Earning

  • /daily — Claim daily rewards (24h cooldown)
  • /work — Work a random job to earn points
  • /rob <user> — Risk robbing another user

Banking

  • /bank deposit <amount> — Deposit into your bank (protected from robbery)
  • /bank withdraw <amount> — Withdraw from your bank
  • /bank balance — View bank and wallet balances
  • Loan system with interest and repayment tracking

Gambling

  • /blackjack — Full blackjack with hit, stand, double
  • /poker — Five-card draw poker
  • /flip — Heads or tails coin flip
  • /dice — Bet on dice rolls
  • /slots — Slot machine with multipliers
NoteAll currency is virtual with no real-world value. Economy can be toggled with /points enable/disable.

Leveling System

Members earn XP by chatting. XP accumulates and unlocks levels with configurable role rewards.

  • Automatic XP — users earn XP for messages (with cooldowns)
  • Level-based roles — auto-assign roles at specific levels
  • /rank [user] — view level, XP, and progress
  • /leaderboard / /lb — server XP leaderboard
  • Admin controls — /xp and /level to manage values
TipEnable/disable with /levels enable/disable. Configure level roles through settings.

Games

Chess

  • Play against another member or the bot AI
  • Full chess rules with move validation
  • Interactive button-based interface for moves, draws, and resignation
  • Board rendered as text with emoji pieces

Poker

  • Five-card draw poker with betting
  • Hold and discard mechanics via buttons

Proxy System

A PluralKit-style proxy system that lets users send messages as different identities using webhooks.

How It Works

  1. Create a proxy member with /proxy create <name>
  2. Set proxy tags (prefix/suffix) — e.g., {text} sends as that member
  3. Set a custom avatar with /proxy avatar <name> <url>
  4. Messages matching your tags are automatically proxied via webhooks

Features

  • Multiple members — create as many proxy identities as you need
  • Groups — organize members into groups
  • Custom avatars — each member can have its own avatar
  • Switch system — /proxysettings switch <member> to auto-proxy all messages
  • Proxy indicator — servers can enable a [proxied] tag for transparency

Cross-Server Global Chat

Link channels across multiple servers so messages relay in real-time.

How It Works

  1. Create a channel with /globalchat create <name>
  2. Other admins join with /globalchat join <code>
  3. Messages relay to all linked channels via webhooks

Management Panel

Use /globalchat manage to open an interactive panel in your DMs. All actions use modal forms — no need to type in chat.

  • Edit Channel — update name, description, visibility, join key, and webhook mode
  • Display Format — customise the name format, message prefix, and message suffix shown on relayed messages
  • Set Rules — configure channel rules shown to members on join
  • Add Moderator — grant a user moderator access to the channel
  • Kick / Ban / Warn / Mute — moderate linked servers directly from the panel
  • Unmute / Clear Warnings — reverse moderation actions
  • View Linked — list all servers currently linked to the channel

Safety Features

  • Attribution — relayed messages prefixed with [GC] (customisable)
  • Rate limiting — 3-second per-user cooldown
  • Content limits — 2000 chars, max 5 attachments
  • Server muting — mute specific servers from sending or receiving
  • Moderation — owners and moderators can kick, ban, warn, or mute servers

Ticket System

A support ticket system for managing user inquiries and issues.

  • Ticket panels — create embed panels with a button to open tickets
  • Auto-creation — clicking creates a private channel for user and staff
  • Close & archive — tickets can be closed, archived, or deleted
  • Transcript logging — ticket conversations can be logged for review

Auto-Moderation

Automated message filtering to keep your server clean.

  • Spam detection — configurable thresholds for rapid messages
  • Link filtering — block or whitelist specific domains
  • Mention spam — limit mass mentions
  • Caps filter — detect excessive caps
  • Invite blocking — remove Discord invite links
  • Word blacklist/whitelist — custom word filtering
  • Bypass permission — users with automod.bypass are exempt

Configure with /automod config and /antispam.

Comprehensive Logging

Log server events to designated channels for transparency and record-keeping.

  • Message logs — edits, deletions, and bulk deletes
  • Moderation logs — bans, kicks, mutes, warns
  • Join/leave logs — member join, leave, and role changes
  • Manual logs — /log for staff to create manual entries
  • Selective log types — configure which events go to which channels with /logchannel

Welcome System

  • Welcome messages — customizable messages when members join
  • Auto-roles — automatically assign roles to new members
  • Variable support — placeholders for user names, server names, member counts

Configure with /welcome.

Suspicious Account Detection

Automated system that analyzes new members and flags potentially suspicious accounts.

  • Account age analysis — flags very new accounts
  • Pattern detection — identifies bot-like username patterns
  • Risk scoring — multi-factor suspicion level calculation
  • Admin notifications — alerts sent to designated channels
  • Validate/invalidate — staff can mark accounts as safe or suspicious
  • Global masterlist — cross-server suspicious user tracking

Role Persistence

When a member leaves and rejoins, their roles are automatically restored. Configurable expiry period. Enable/disable with /rolepersistence.

Reaction Roles

Let members self-assign roles by reacting to messages. Create panels with /reactionrole. Supports custom and Unicode emojis.

Pride & Inclusivity

  • /pride <flag> — apply pride flag overlays to your avatar
  • /flags — list all available pride flags
  • /pronouns — set pronoun roles for yourself

Server Backups

Create and restore snapshots of your server configuration with /backup create and /backup restore.

Privacy & Data

  • /privacy — view the privacy policy in Discord
  • /deletedata — permanently delete all your personal data
